# EJEMPLO APP CON NEXT-AUTH

## Instalación

Copiar el archivo .env.example y renombrarlo a .env.local

```bash
cp .env.template .env.local
```

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

## Descripción

Este es un ejemplo de una aplicación con Next.js y Next-Auth para la autenticación de usuarios.

## Tecnologías

- Next.js
- Next-Auth
- MongoDB
- TailwindCSS
